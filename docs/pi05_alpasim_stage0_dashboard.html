Data Used

  • Raw source family: NVIDIA PhysicalAI Autonomous Vehicles, paired to matching NuRec simulator scenes.
  • Manifest constraint: exactly 5 clips, with at least one left_turn and one right_turn.
  • Per-clip target duration: 20.0 s.
  • Expected sample rate: 10 Hz.
  • Total local dataset used for training: 5 episodes, 545 total frames.
  • State history uses speed, yaw rate, and acceleration windows.
  • Route corridor length: 32 future points.
Training flow

Raw AV + NuRec
  -> dataset build
  -> LeRobot local repo (local/stage0_av_driving)
  -> PI0.5 LoRA fine-tune
  -> checkpoint /pi05_stage0_av/1000
  -> AlpaSim external driver

Runtime flow

Sensorsim cameras + route
  -> PI0.5 driver
  -> action chunk (delta_s, delta_yaw, speed)
  -> kinematic bridge + clamps
  -> controller + physics
